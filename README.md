A community-driven library of reusable, secure, and scalable wireless communication modules for public-sector and commercial applications.

## Features
- 📡 PSK/QAM modulators and demodulators
- 🔒 Cryptographic primitives for wireless links
- 🚀 Optimized for real-time SDR applications
- 🌍 Luganda language support for Ugandan developers
