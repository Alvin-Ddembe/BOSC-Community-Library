# Contributing to BOSC Community Library

## How to Contribute
1. Fork the repository
2. Create a feature branch (`git checkout -b feature/amazing-modulation`)
3. Commit your changes (`git commit -m "Add amazing modulation scheme"`)
4. Push to the branch (`git push origin feature/amazing-modulation`)
5. Open a Pull Request

## Code Style
- Python: Follow PEP 8
- Add comments for wireless/signal processing logic
- Include unit tests for any new modulation or coding scheme

## Pull Request Process
- Update the README.md if needed
- Ensure all tests pass
- Your PR will be reviewed by maintainers within 3-5 days

## Questions?
Open an issue with the "question" label.
